# Viewshed-Analysis-with-Python

This project demonstrates how to perform a visibility analysis using Python based on elevation data. The goal is to determine whether the generated points are visible from a specific observation point(POI) by analyzing terrain heights.

Rasterio is used on core logic of visibility calculation directly on Pyhton environment.

The objective of this project is to:

- Implement a visibility analysis algorithm in Python.

- Use elevation data to determine line-of-sight visibility between points.

- Demonstrate how such an analysis works without GIS-specific libraries such as GDAL.
